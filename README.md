# Simple Portfolio Template
A simple website portfolio template and tutorial for computer scientists and software engineers, designed to introduce beginners to HTML and CSS!

## Using This Template

1. To edit this template, fork this repo! (Click the fork button in the top right corner.)
2. Clone the repo.
    * Make sure you have [Git downloaded](https://git-scm.com/install/).
    * Click the green code button in the top right corner and copy the HTTPS URL.
    * In your command line, navigate to your desired folder and `git clone <url>`
3. Check that you are in the main branch using `git branch`
4. Customize the files however you want! Examples:
    * Follow the TODO comments to replace the placeholders with your info
    * Change the background, fonts, colors, and other styles
    * Add a new section for personal projects
5. Make sure to commit your changes and push to the global repo.
    * `git add .`
    * `git commit -m "<description-of-changes>"`
    * `git push`

## Hosting on GitHub Pages

1. Navigate to your forked repository.
2. Click Settings at the right of your top menu bar.
3. Go to the Pages tab in the left menu.
4. Under the Build and deployment heading, make sure "Deploy from a branch" is selected.
5. Under Branch, select "main" for your branch and "/ (root)" for your folder, and hit Save.
6. GitHub will generate a URL to your site.
